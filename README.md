# ScutoidOS 2
+ Limine Bootloader

### System Requirements
+ 4 MB of ram
+ 8 MB storage
### Building Instructions

On a linux computer
+ Install xorriso
+ Open vscode and cd to the extracted ScutoidOS folder
+ Download qemu.sh and run it (sh "PATH")
+ Rename the file ending to .iso (Delete the ending, press enter, and then add .iso again)
+ Boot
